# Yarukoto 画面設計書

## 1. デザイン方針

### 1.1 デザインスタイル
- **スタイル**: カード型・モダン
- **特徴**: 影やボーダーを使った視覚的に区切られたデザイン
- **余白**: 適度な余白で読みやすさを確保

### 1.2 カラーテーマ
| 用途 | 色 | カラーコード |
|------|-----|-------------|
| プライマリ | オレンジ | #F97316 |
| プライマリ（ホバー） | ダークオレンジ | #EA580C |
| 背景 | ライトグレー | #F9FAFB |
| カード背景 | ホワイト | #FFFFFF |
| テキスト（主） | ダークグレー | #111827 |
| テキスト（副） | ミディアムグレー | #6B7280 |
| ボーダー | ライトグレー | #E5E7EB |
| 成功（完了） | グリーン | #10B981 |
| 警告（期限超過） | レッド | #EF4444 |
| やらない | グレー | #9CA3AF |

### 1.3 タイポグラフィ
- **フォント**: Noto Sans JP
- **見出し**: 太字、サイズ大きめ
- **本文**: 通常ウェイト、読みやすいサイズ

## 2. 共通コンポーネント

### 2.1 ヘッダー
```
┌─────────────────────────────────────────────────┐
│  [≡]  Yarukoto              [検索] [設定]       │
└─────────────────────────────────────────────────┘
```
- 左: ハンバーガーメニュー（カテゴリ管理へのアクセス）
- 中央: アプリ名/現在の日付
- 右: 検索アイコン、設定アイコン

### 2.2 日付ナビゲーション
```
┌───────────────────────────────────────────────────────┐
│  [<]     2024年1月15日（月）今日      [>]  [今日] [📅] │
└───────────────────────────────────────────────────────┘
```
- 左右矢印: 前日/翌日へ移動（スワイプでも可）
- 中央: 現在表示中の日付（今日の場合は「今日」バッジ表示）
- 右端: 今日ボタン（今日以外の日付表示時のみ表示、タップで今日に戻る）、カレンダーアイコン（日付ピッカー表示）

### 2.3 タスクカード
```
┌─────────────────────────────────────────────────┐
│ [□] タスク名                                    │
│     📅 1/15  🏷️ カテゴリ名  ⚡ 高               │
└─────────────────────────────────────────────────┘
```
- 左: チェックボックス（完了/未完了切り替え）
- 中央上: タスク名
- 中央下: 予定日、カテゴリ、優先度（設定されている場合のみ表示）

#### タスクカードの状態別表示
| 状態 | 表示スタイル |
|------|-------------|
| 未完了 | 通常表示 |
| 完了 | チェック済み、テキストに取り消し線、背景薄いグリーン |
| やらない | テキストグレー、背景薄いグレー、理由表示（あれば） |
| 期限超過 | 左側にレッドのアクセントライン |

#### 左スワイプ操作メニュー
```
スワイプ後:
┌─────────────────────────────────────────────────────────────┐
│ [□] タスク名              │ [編集] [やらない] [削除] │
└─────────────────────────────────────────────────────────────┘
                                 青      グレー      赤
```

### 2.4 タスク入力欄（下部固定）
```
┌─────────────────────────────────────────────────┐
│ [+] 新しいタスクを入力...        [📅今日] [▶]  │
└─────────────────────────────────────────────────┘
```
- 左: プラスアイコン（フォーカス時に非表示）
- 中央: テキスト入力欄
- 右: 日付選択ボタン（デフォルト: 今日）、送信ボタン
- Enter キーで送信＆連続入力可能

5. **やらない**: 今日「やらない」にしたタスク（折りたたみ可能）

## 5. インタラクション仕様

### 5.1 スワイプ操作

#### 日付ナビゲーション（左右スワイプ）
- **右スワイプ**: 前日へ移動
- **左スワイプ**: 翌日へ移動
- **アニメーション**: 0.3秒のスライドアニメーション

#### タスク操作（左スワイプ）
- **左スワイプ**: 操作メニュー表示
  - 編集ボタン（青）
  - やらないボタン（グレー）
  - 削除ボタン（赤）
- **スワイプ距離**: 100px以上で確定
- **自動クローズ**: 他の場所タップで閉じる

### 5.2 タスク作成フロー

1. 入力欄にテキスト入力
2. （任意）日付ボタンタップで予定日変更
3. Enterキーまたは送信ボタンでタスク作成
4. 入力欄はクリアされ、次のタスク入力可能

### 5.3 タスク完了フロー

1. チェックボックスタップ
2. 即座に完了状態に変更（取り消し線、背景色変化）
3. 完了セクションに移動（アニメーション）
4. 再度タップで完了解除可能

### 5.4 フィードバック

| 操作 | フィードバック |
|------|---------------|
| タスク作成 | 軽い振動 + リストにフェードイン |
| タスク完了 | チェックマークアニメーション + 位置移動 |
| タスク削除 | 左にスライドアウト |
| エラー | 赤いトースト通知 |
